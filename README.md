# 🎓 Student Performance Prediction System

## 📌 Project Overview
The Student Performance Prediction System is a Machine Learning project developed to predict a student's final academic grade based on factors such as study time, failures, absences, and previous examination scores.

The project uses a Linear Regression model from Scikit-Learn and provides an interactive user interface built with Streamlit for real-time predictions.

---

## 🚀 Features
- Predicts student final grades (G3)
- Interactive web application using Streamlit
- Real-time prediction based on user inputs
- Data preprocessing and model training using Scikit-Learn
- Easy-to-use interface for educational analysis

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Matplotlib

---

## 📊 Machine Learning Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Model Evaluation
6. Real-Time Prediction using Streamlit

---

## 📈 Model Performance
- Algorithm: Linear Regression
- Accuracy (R² Score): 78%
- Dataset: Student Performance Dataset

---

## 🎯 Project Outcome
This project helps identify student performance trends and provides grade predictions based on academic and behavioral factors. It demonstrates practical applications of Machine Learning in the education sector.
